# 🚀 Deploy Article Website NOW - 2 Minute Setup

## ⚡ Quickest Way (Copy & Paste)

### Step 1: Open This Link
```
https://vercel.com/new?repo=ikrigel/south-lebanon-map
```

### Step 2: Change Root Directory
When you see the import dialog:
- Find the **Root Directory** field
- **Change from:** `docs-website`
- **Change to:** `article-website`

### Step 3: Click Deploy
Click the **Deploy** button and wait 1-2 minutes.
